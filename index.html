üß† Key Projects

  • GloCal-ResSAMNet: A hybrid segmentation model combining Segment Anything Model (SAM), GNN-based vessel topology learning, and ResUNet++ for precise artery, vein, and capillary segmentation in OCTA images.
  • ASDCBNet-WWTP: An attention-enabled CNN-BiLSTM architecture developed for real-time multi-parameter monitoring and forecasting of sewage treatment plants (STPs), ensuring optimized water management.
  • SatelliteSR-GAN: GAN-based satellite image super-resolution using RRDB backbone, Swin Transformer encoder, and hybrid attention for reconstructing high-resolution Earth imagery from low-res inputs.
  • GrapeLeaf-MaskRCNN: Grape leaf disease classification using Mask R-CNN for precise region segmentation, handcrafted texture features, and a Swin Transformer backbone enhanced with CBAM for attention learning.
  • CKD-EnsembleNet: An ensemble deep learning framework for chronic kidney disease detection using stacked autoencoders, CapsNet, GB-DNN classifiers, and LSTM for feature fusion and diagnosis prediction.
  • Hybrid HBF-SIC-DRL: Deep Reinforcement Learning-based Hybrid Beamforming + Successive Interference Cancellation architecture for 5G MU-MIMO environments, targeting low-latency communication.
  • Multimodal Biometric Auth: Deep multimodal biometric authentication system using ECG and lip-motion signals with hybrid deep models including 1D CNNs, GRUs, and attention modules for robust identity recognition.
  • SAM-OCTA500: Fine-tuned Segment Anything Model (SAM) with LoRA on the OCTA500 dataset for capillary and vessel segmentation, optimized for few-shot medical imaging tasks.
